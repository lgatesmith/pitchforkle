# Pitchforkle

Guess the Pitchfork rating from the album cover alone!

## About

Pitchforkle is a daily music game where players try to guess the Pitchfork rating of an album based solely on its cover art. Think Wordle, but for music critics!

## Development

This project is built with React, TypeScript, and Vite.

### Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## How to Play

1. View an album cover
2. Guess the Pitchfork rating (0.0 - 10.0)
3. Get feedback on how close you are
4. Try to guess the exact rating in as few attempts as possible!

## Tech Stack

- React
- TypeScript
- Vite
